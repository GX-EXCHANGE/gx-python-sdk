# GX Exchange Python SDK

Official Python SDK for interacting with the GX Exchange API. Provides a complete interface for programmatic trading, account management, and market data retrieval on the GX Chain network.

## Overview

- Full-featured Python client for the GX Exchange REST and WebSocket APIs
- Supports order placement, cancellation, and account state queries
- Market data retrieval including order books, trades, and funding rates
- Compatible with both mainnet and testnet environments
- EIP-712 signing for secure transaction authentication

## Requirements

- Python 3.10+
